# Proyecto tienda PostgreSQL

Ejecuta los archivos en orden.

1. `02-crear-schemas.sql`
2. `03-crear-tablas.sql`
3. `04-crear-relaciones.sql`
4. `05-insertar-datos-prueba.sql`
5. Consultas `06` a `09`
6. Objetos de negocio `10` a `13`

El proyecto es educativo. Para producción se deben añadir migraciones, auditoría completa, pruebas, roles adecuados, validación de moneda, impuestos y manejo de concurrencia en el procedimiento de compra.
