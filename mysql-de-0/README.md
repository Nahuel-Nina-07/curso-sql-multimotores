# MySQL desde cero

Curso práctico, ordenado y completo para aprender MySQL usando MySQL Server y MySQL Workbench.

## Regla de estudio
1. Lee los archivos `.md` antes de ejecutar los `.sql` del mismo módulo.
2. Ejecuta los scripts en el orden numérico.
3. Resuelve cada ejercicio antes de mirar la solución incluida.

## Convenciones
- Motor de tablas: `InnoDB`.
- Juego de caracteres: `utf8mb4`.
- IDs: `INT UNSIGNED AUTO_INCREMENT`.
- En MySQL, `DATABASE` y `SCHEMA` son equivalentes.
- No uses la cuenta `root` desde una aplicación real.
